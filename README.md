# Iris – Interactive Social & Marketplace Platform

## Project Overview
**Iris** is a web platform that allows users to share statuses, chat in real-time, buy and sell items, order food, and track deliveries or public transport on a live interactive map. It seamlessly integrates social networking, marketplace, and location-based services.

## Key Features
- **Social:** Status updates, chat, reactions, friend system
- **Marketplace:** Buy/sell items, favorites, checkout, promotions
- **Food & Delivery:** Browse food shops, place orders, live delivery tracking
- **Commute & Map:** Live vehicle tracking, route planning, ETA notifications
- **Security & Admin:** Role-based access, input validation, admin dashboard
